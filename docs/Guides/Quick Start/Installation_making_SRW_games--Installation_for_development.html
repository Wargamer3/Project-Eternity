Installation for development


1. Install Visual Studio
1) Visual studio should install all the dependencies required but if not just ensure you have .NET 6.1
2. Install XNA SDK
1) Flatredball already has everything you need here.
2) If the website is down, you can install it from the engine repo here.
3) If something goes wrong you can also refer to the Azure compiler here for hints on where to copy files.
3. The engine may be downloaded from this location (Optional)
4. If you want the SRWE assets, they can be found here. Clone it under the Project Eternity\Project EternityContent folder and follow the instructions. (optional)
