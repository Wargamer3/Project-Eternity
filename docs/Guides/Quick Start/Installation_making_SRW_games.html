Installation making SRW games


1. Be sure to have installed the latest Microsoft XNA Framework Redistributable: http://www.microsoft.com/download/en/details.aspx?id=20914
2. And the .NET Framework 4.6.1 https://dotnet.microsoft.com/download/dotnet-framework/net461
3. Download the SRWE build https://www.dropbox.com/s/phh7y23ko2rgpq8/SRWE%20public%20demo.zip?dl=0
