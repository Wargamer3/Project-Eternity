Making a game


It’s already a pretty colossal task to make a game but making a game in an unknown engine is even harder. To help with that we’ll go through the basic asset creation needed to make a game. For more in depth information you can check the official guide here.

This tutorial will teach you about:      
1. Creating Unit/Pilot/Attacks
2. Creating a map
3. Creating dialogues
4.  Creating cutscenes
5. Creating animations (Optional)

After doing all of this you’ll have a fully working stage and you’ll be able to start tying them together to make a proper story.
The order of asset creation is not important, you can easily split the work between multiple people without getting blocked anywhere if you use placeholders where needed.

This tutorial will follow the creation of a demo for SRWE so you can follow it by using any demo.
